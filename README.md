## Description

CanvasHack is a Chrome browser extension designed to bypass Canvas's built-in quiz activity tracking. It hides tab switches, window changes, split-screen usage, background activity, and other actions that would normally appear in the Canvas quiz logs as suspicious (e.g., "Left quiz tab", "Stopped viewing the Canvas quiz-taking page").


### Key Features

- **Undetectable tab & window switching** ‚Äî Flip between tabs or apps freely; Canvas sees only normal quiz activity.
- **No suspicious logs** ‚Äî Blocks or spoofs visibility events so "Left quiz tab" or "Stopped viewing" never appears.
- **Answer saving & smart second attempts** ‚Äî Remembers your best answers across attempts and auto-fills them for higher scores.
- **Kiosk mode spoofing** ‚Äî Makes the quiz appear locked/secure on the surface while you retain full browser freedom.
- **Seamless & lightweight** ‚Äî Easy one-click install, no performance impact.
- **Works on all Canvas domains** ‚Äî Compatible with school, university, and institution Canvas instances.
- **Live real-exam demos available** ‚Äî Watch proof videos showing clean logs vs. flagged logs.

Follow these steps to install CanvasCracked:

### 1. Download the Extension
- Download the repository as a ZIP file from the releases 
- Extract (unzip) the contents to a folder on your computer

### 2. Open Chrome Extensions Page
- Open Google Chrome
- Navigate to: chrome://extensions

### 3. Enable Developer Mode
- In the top-right corner, toggle **Developer mode** ON
